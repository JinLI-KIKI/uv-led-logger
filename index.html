Connect Google Sheets

Follow these steps once to sync all logs to a spreadsheet accessible by the whole lab.

Step 1 — Create the spreadsheet
Open Google Sheets, create a new file named UV-LED Log. Add two sheets: UsageLog and ParamLog.
Step 2 — Add the Apps Script
In the spreadsheet: Extensions → Apps Script. Delete all existing code and paste the script below. Click Save.
Step 3 — Deploy as Web App
Click Deploy → New deployment → Web app. Set "Execute as: Me" and "Who has access: Anyone". Click Deploy. Copy the URL.
Step 4 — Paste the URL
Go back to the Usage tab, paste the URL into the "Apps Script Web App URL" field. All future sessions will sync automatically.
